smolOS - XIAO RP2040 Installation Guide

Install MicroPython

Download latest firmware from micropython.org/download/ for the RP2040.

  • Long press boot button on the XIAO for firmware update mode
  • Plug into the USB while still pressing
  • Push after 1 sec
  • New device should popup
  • Mount the drive
  • Copy firmware to it
  • XIAO will restart itself
