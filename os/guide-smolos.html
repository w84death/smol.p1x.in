smolOS - Installation Guide

Install MicroPython

Your microcontroller needs to have latest Python firmware installed.

  • Guide for XIAO RP2040
  • Guide for ESP8266

After that you can copy smolOS files.

Install smolOS

As main script:

  • put main system file smolos.py onto the board flash
  • copy main.py (auto-boot)

As separate script that will work with your existing files on the device:

  • do not copy main.py
  • you will need to boot OS by hand each time


Prepare Environment

Do this once (initialization, ampy installation):

$ python3 -m venv venv
$ source venv/bin/activate
$ pip install ampy

Pushing Files

After that each time do this:

$ source venv/bin/activate
$ ampy --port /dev/ttyUSB0 put smolos.py
$ ampy --port /dev/ttyUSB0 put main.py
$ ampy --port /dev/ttyUSB0 put hello.txt

Description:

  • smolos.py is the full smolOS system
  • main.py is a default script that will run at boot and initiate the smolOS
  • hello.txt is just a test file so you have something to play with (optional)
  • put your own files the same way
  • do not overwrite the system boot.py file!

Manual Booting

In MicroPython REPL write:

>>> from smolos import smolOS
>>> os = smolOS()
>>> os.boot()
